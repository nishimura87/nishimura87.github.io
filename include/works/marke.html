Marke ECサイト

スクール卒業試験でECサイトを作成しました。
デザイン〜デプロイまで全てを実施いたしました。
GoogleMapやStripeをAPI連携させ、マップ表示・クレジット決済を可能としています。
ユーザー認証を取り入れ、カート機能にはセッションを利用して購入できます。
管理者は商品やニュースのCRUDが可能です。
UI、UXに力を入れるべく今後も少しずつアップデートしていこうかなと思います。
サーバーはAWSを利用しました。

使用言語など

HTML / css / Laravel / JavaScript / AWS
